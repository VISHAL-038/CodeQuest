Functions and Modules

Discover how to create reusable code with functions and organize your programs using modules. Enhance your programming efficiency and structure.

Defining and Using Functions

Learn how to define functions to encapsulate code into reusable blocks. Understand function parameters, return values, and scope.

# Define a function
def greet(name):
    return f"Hello, {name}!"

# Use the function
print(greet("Alice"))

Working with Modules

Explore how to use modules to organize your code into separate files. Learn about importing modules, creating your own, and using built-in libraries.

# In file: my_module.py
def add(a, b):
    return a + b

# In another file
import my_module
print(my_module.add(2, 3))
