Data Structures

Dive into essential data structures like lists, dictionaries, sets, and tuples. Learn how to store and manipulate data effectively in Python.

Lists

Lists are ordered collections of items. You can create a list, add and remove items, and access elements by their index.

Example Code

# Creating a list
my_list = [1, 2, 3, 4, 5]

# Adding an item
my_list.append(6)

# Removing an item
my_list.remove(3)

# Accessing elements
first_item = my_list[0]

Dictionaries

Dictionaries store data as key-value pairs. Learn how to create dictionaries, access values using keys, and modify entries.

Example Code

# Creating a dictionary
my_dict = {'name': 'Alice', 'age': 25}

# Accessing values
name = my_dict['name']

# Adding a new key-value pair
my_dict['city'] = 'New York'

# Removing a key-value pair
del my_dict['age']

Sets

Sets are collections of unique items. Learn how to create sets, perform set operations like union and intersection, and manage set data.

Example Code

# Creating a set
my_set = {1, 2, 3}

# Adding an item
my_set.add(4)

# Removing an item
my_set.discard(2)

# Set operations
another_set = {3, 4, 5}
union_set = my_set.union(another_set)
intersection_set = my_set.intersection(another_set)

Tuples

Tuples are immutable sequences. They are useful for storing fixed collections of items. Learn how to create and access tuple elements.

Example Code

# Creating a tuple
my_tuple = (1, 2, 3)

# Accessing elements
first_element = my_tuple[0]

# Tuples are immutable
# my_tuple[1] = 4  # This will raise an error
