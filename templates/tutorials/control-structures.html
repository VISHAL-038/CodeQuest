Control Structures

Learn how to control the flow of your program with loops and conditionals. Mastering these concepts will help you solve complex problems more efficiently.

Introduction to Loops

Loops are used to execute a block of code repeatedly. In this section, you'll learn about different types of loops:

  • For Loops: Used to iterate over a sequence (like a list, tuple, or string) or range of numbers.
  • While Loops: Continue to execute as long as a specified condition is true.

Example Code

# For loop example
for i in range(5):
    print(i)

# While loop example
count = 0
while count < 5:
    print(count)
    count += 1

Introduction to Conditionals

Conditional statements are used to execute code based on certain conditions. You will learn about:

  • If Statements: Execute code if a condition is true.
  • Else Statements: Execute code if the condition is false.
  • Elif Statements: Check additional conditions if the previous conditions are false.

Example Code

# If-Else example
x = 10
if x > 5:
    print("x is greater than 5")
else:
    print("x is 5 or less")

# Elif example
y = 10
if y > 10:
    print("y is greater than 10")
elif y == 10:
    print("y is equal to 10")
else:
    print("y is less than 10")
